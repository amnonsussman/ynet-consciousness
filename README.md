# Public Consciousness Space – Ynet Comment Community  
**מרחב התודעה הציבורית – קהילת המגיבים של Ynet**

This repository hosts an interactive visualization of Israel’s online public discourse,  
analyzed through the PCS (Public Consciousness Space) protocol.  
The current visualization presents four contemporary narratives:
- The bereaved father confronting Netanyahu  
- The “Prosecutor General affair” (פצרית)  
- “Guy Peleg the traitor”  
- “Ben Gvir the bully”

Each cluster represents a major ideological bloc, with metrics for engagement, polarity, and moral tone.
